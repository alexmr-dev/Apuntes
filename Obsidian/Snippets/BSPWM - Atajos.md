### BSPWM

- Crear una nueva ventana: `Win + Enter`
- Cerrar una ventana: `Win + W`
- Navegar entre las diferentes ventanas (espacios): `Win + {1,2,3...}`
	- Mover una terminal o aplicación a otra ventana (espacio): `Win + Shift + {1,2,3...}`
	- Teniendo varias terminales/aplicaciones en una misma ventana, mover un proceso a otra: `Win + Shift + ⮀`
- Desacoplar una ventana: `Win + S`
- Teniendo la ventana desacoplada_
	- Arrastrarla: `Win + Click Izquierdo`
	- Modificar su tamaño: `Win + Click Derecho`
	- Acoplar de nuevo: `Win + T`
- Crear una nueva ventana con preselectores: `Win + Ctrl + Alt + ⮀
![[Pasted image 20250217220142.png|500]]
- Redimensionar alguna pestaña: `Win + Shift + ⮀
- Maximizar una ventana: `Win + F`
- Abrir el Rofi: `Win + D`
- Abrir Firefox: `Win + Shift + F`
- Reiniciar bspwm: `Win + Alt + R`
- Reiniciar todo (reboot): `Win + Alt + Q`

### Módulos

- Establecer un objetivo: `target "IP - Nombre"`
- Limpiar el objetivo: `target reset`

### Kitty

Teniendo una terminal abierta:
- Dividir a la mitad la terminal, y así tener dos: `Ctrl + Shift + Enter`
- Cambiar el tamaño de alguna de ellas: `Ctrl + Shift + R`

![[Pasted image 20250217215201.png|400]]

- Hacer zoom en alguna de ellas: `Ctrl + Shift + Z` (Para quitar el zoom es igual)
- Crear una nueva pestaña: `Ctrl + Shift + T
![[pestanyas.png]]
- Renombrar las pestañas: `Ctrl + Shift + Alt + T`
- Moverte entre pestañas:
	Hay dos opciones para esto:
	1. `Ctrl + Tab`
	2. `Ctrl + Shift + ⮀`
- Cerrar una pestaña: `Ctrl + Shift + W`
